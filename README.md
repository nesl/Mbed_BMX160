# Mbed_BMX160
Drivers and Mbed Code for Bosch BMX160 9DoF Absolute Orientation Sensor (https://www.bosch-sensortec.com/products/motion-sensors/absolute-orientation-sensors/absolute-orientation-sensors-bmx160.html)
